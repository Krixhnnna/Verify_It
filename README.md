
# Verify It - Counterfeit Item Detector

A platform that checks if a product is real or fake. Prevents counterfeits with simple and secure verification.

<img width="815" height="501" alt="Project Banner" src="https://github.com/user-attachments/assets/01308bf2-efa0-4caa-996d-f1b84e59f162" />

## 🚀 Features

- **Real-time Verification**: Instantly check product authenticity.
- **Secure Database**: Robust tracking of verified items using PostgreSQL.
- **Responsive UI**: Optimized for both desktop and mobile devices.
- **Traffic Analytics**: Built-in tracker for monitoring verification attempts.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Deployment**: Vercel

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Krixhnnna/Verify_It.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Database Setup**:
   - Create a PostgreSQL database.
   - Run the schema provided in `database.sql`.
   - Update your environment variables.

4. **Start the server**:
   ```bash
   npm start
   ```
