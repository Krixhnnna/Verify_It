📖 About the Project

How VerifyIt Works

VerifyIt is an anti-counterfeit system that helps consumers and businesses verify the authenticity of products using unique serial numbers stored in a secure PostgreSQL database.

The Process

01

Enter Product Details

Users enter the product serial number or unique code available on the product packaging to begin the verification process.

02

Consumer Checks

The buyer enters the serial number found on the product or its packaging into our website.

03

Database Lookup

Our Express server queries the PostgreSQL database for a matching serial number instantly.

04

Result Returned

If the serial exists → Genuine. If not found → Counterfeit alert is shown to the user.

Technology Stack

This project is built with simple, industry-standard tools that are easy to understand:

🟢 Node.js — Runtime 🚂 Express.js — Web Server 🐘 PostgreSQL — Database 🎨 JS — Routes and Server 🎯 CSS — Styling

Why This Matters

Counterfeit products cost the global economy over $500 billion annually. They are dangerous (fake medicines, electronics), harm brand reputations, and cheat consumers. A simple serial-number verification system can significantly reduce counterfeiting.
